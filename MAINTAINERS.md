## Responsibilities

Maintainers are responsible for:
- Reviewing and merging Pull Requests
- Managing releases and versioning
- Responding to Issues and Discussions
- Setting the technical direction of the project

## Becoming a Maintainer

If you are interested in becoming a maintainer, please start by contributing to the project via Pull Requests and engaging in technical discussions.
